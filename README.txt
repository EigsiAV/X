Titre de Projet : Conception et Développement d'un Système Temps Réel et Multi-tâche : Gestion Automatisée de Téléviseurs dans une Maison Intelligente

Résumé du projet ( Abstract) : Ce projet vise à développer un système automatisé de gestion des téléviseurs pour une maison intelligente. Le système permettra de contrôler l'état de marche/arrêt de trois téléviseurs, en respectant certaines contraintes définies.

Code et Simulation du projet (Lien Wokwi) : https://wokwi.com/projects/399612266411642881
